# S&S Watch Shop (Frontend, React + Vite + Tailwind)

A sleek, production-ready frontend for a watch shopping site with:
- Product grid with wishlist
- Product details page
- Cart with quantity edit & remove
- Checkout with address, payment (UPI/Card/COD) & coupons
- Order success screen
- Modern UI with Tailwind + Framer Motion

## Quick Start

```bash
# 1) Extract and install
npm install

# 2) Start dev server
npm run dev

# 3) Build for production
npm run build
npm run preview
```

## Tech
- React 18 + Vite
- TailwindCSS 3
- React Router 6
- Framer Motion
- Lucide React Icons

## Customise branding
- Update logo in `src/components/Logo.jsx`
- Change site name in `src/components/Navbar.jsx`
- Edit products in `src/data/products.js`
